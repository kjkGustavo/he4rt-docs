- 1 : Crie um projeto em sua IDE escolhida

- 2 : Digite o seguinte código:

```cpp{0}
#include <iostream>
int main ()
{
    std::cout << "Hello World" << std::endl;
}
```

- 3 : Compile e execute

- Pronto! Você fez o seu primeiro Hello World!
